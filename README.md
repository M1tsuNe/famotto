# Famotto

家族向けの情報共有アプリです。

## 概要

家族間で予定や情報を共有できるWebアプリです。

## 開発人数

2名

## 担当箇所

- ログイン機能
- ユーザー認証
- DB設計
- 一部画面実装

## 使用技術

| 項目 | 技術 |
|------|------|
| Frontend | Next.js |
| Backend | Spring Boot |
| Database | PostgreSQL |
| Infrastructure | Docker |

## 画面イメージ

![トップ画面](images/top.png)

## 起動方法

```bash
git clone <repository-url>
npm install
npm run dev
```

## 工夫した点

- JWT認証を利用
- レスポンシブ対応
- 家族単位でデータを管理できる設計
